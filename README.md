# NORA

## Description

+ Nora is a REST API for relational database management, for multiple applications.
+ Starter project for node.js.

Featured:

* Node.js
* Gulp
* Bower
* Jade
* SASS
* CoffeeScript

## Quick Install

1. npm install
2. gulp
3. Happy coding!

## Hosting

1. Just send the file **package.json** and the folder **server** (Node.js)
